# Research

## Objetivo de Research

Comprender cómo los usuarios buscan información sobre el universo de One Piece, qué dificultades encuentran durante ese proceso y qué características consideran importantes para consultar personajes, tripulaciones, frutas del diablo y arcos de forma rápida, organizada y visual.

La investigación busca identificar oportunidades para diseñar una experiencia que reduzca el tiempo de búsqueda y facilite la navegación entre los diferentes elementos del universo de One Piece.

---

# Hipótesis

Antes de iniciar la investigación se plantean las siguientes hipótesis:

- Los usuarios buscan principalmente personajes antes que cualquier otro contenido.
- Los usuarios necesitan navegar fácilmente entre personajes, tripulaciones, frutas del diablo y arcos sin realizar múltiples búsquedas.
- La información distribuida en varias páginas incrementa el tiempo necesario para encontrar datos específicos.
- Una interfaz visualmente organizada facilita la comprensión de las relaciones entre personajes y eventos.
- Un buscador con filtros mejoraría significativamente la experiencia de consulta.

Estas hipótesis deberán validarse mediante investigación con usuarios y no deben considerarse hechos.

---

# Método de investigación

Para este proyecto se propone una estrategia en dos etapas.

## Primera etapa

Entrevistas semiestructuradas con personas que hayan visto o estén viendo One Piece para comprender:

- Cómo buscan información actualmente.
- Qué páginas utilizan.
- Qué problemas encuentran.
- Qué información consultan con mayor frecuencia.

## Segunda etapa

Encuesta dirigida a una muestra mayor de usuarios para validar si los patrones encontrados durante las entrevistas se repiten entre distintos perfiles.

La combinación de entrevistas y encuestas permitirá obtener información cualitativa y cuantitativa que respalde las decisiones de diseño.
