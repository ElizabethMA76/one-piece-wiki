# 01 - Definición del problema

## Proyecto

One Piece Wiki

## Contexto

One Piece es una historia extensa con muchos personajes, tripulaciones, frutas del diablo y arcos narrativos. Para una persona nueva o incluso para un fan que desea consultar información rápidamente, puede ser difícil encontrar datos organizados de manera clara y visual.

## Problema

Los usuarios interesados en One Piece pueden tener dificultad para encontrar información relevante de forma rápida y ordenada, ya que los datos suelen estar distribuidos en diferentes sitios, con estructuras extensas o poco visuales.

Esto puede provocar que el usuario tarde más tiempo en ubicar personajes, tripulaciones, frutas del diablo o arcos importantes de la historia.

## Impacto en la experiencia

La búsqueda de información puede volverse lenta, confusa o poco atractiva, especialmente para usuarios nuevos que todavía no conocen bien la relación entre personajes, tripulaciones y eventos principales.

## Criterio de calidad

El problema se define desde la experiencia del usuario y no desde una solución técnica. La propuesta aún no asume una funcionalidad específica, sino que identifica una dificultad actual observable.
